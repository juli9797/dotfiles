# D o t f i l e s

My dotfiles backup.
I use stow to symlink them to the `.config` folder.

**Installation**

Dependencies:
* stow

Clone this Repo into `~/`

**Usage**

```
[jln:~/dotfiles]$ # install polybar dotfiles
[jln:~/dotfiles]$ stow polybar
[jln:~/dotfiles]$ # remove polybar dotfiles
[jln:~/dotfiles]$ stow -D polybar
```
